100 This template is an interface for the general purpose I/O pin hardware. \n
101 
102 It provides:\n
103 - Static pin mapping in conjunction with the configuration settings.
104 - Setting pin characteristics e.g. pull-ups, drive strength etc
105 - Changing the pin direction
106 - Setting outputs to high or low level
107 - Polling input value
108 - Polarity (\ref USBDM::ActiveHigh or \ref USBDM::ActiveLow) is selected when instantiated

218 This template is an interface for the ADC input pins. \n
219 
220 It provides:\n
221 - Static pin mapping in conjunction with the configuration settings. (Some ADC pins have fixed mapping.)

225 - Interrupt driven operation is also supported through a callback if enabled in the configuration.

228 <em>It cannot be instantiated.</em>

316  <b>Usage - PWM</b>
317 @code
318  // Open USBDM namespace.
319  // This allows access to USBDM classes and methods without the USBDM:: prefix.
320  using namespace USBDM;
321 
322  // Initialise the timer with initial alignment
323  // This affects all channels of the FTM
324  Ftm0::configure(FtmMode_LeftAlign);
325 
326  // Set timer period in ticks
327  // This affects all channels of the FTM
328  // Timer pre-scaler is not changed
329  Ftm0::setPeriodInTicks(500);
330 
331  // Set timer period in microseconds
332  // This affects all channels of the FTM
333  // Timer pre-scaler (tick rate) will be adjusted to accommodate the required period
334  Ftm0::setPeriod(125*us);
335 
336  // Use FTM0 channel 3
337  using PwmOutput = Ftm0Channel<3> ;
338 
339  // Set channel to generate PWM with active-high pulses
340  PwmOutput::enable(ftm_pwmHighTruePulses);
341 
342  // Set duty cycle as percentage
343  PwmOutput::setDutyCycle(34);
344 
345  // Set duty cycle as percentage (float)
346  PwmOutput::setDutyCycle(12.25*percent);
347
